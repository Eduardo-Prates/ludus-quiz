# Ludus

> **Ludus** é uma plataforma web de quizzes em tempo real altamente interativa e gamificada. Projetada para ser rápida, fluida e intuitiva, a aplicação permite que apresentadores criem salas de perguntas e participantes respondam de forma síncrona diretamente de seus dispositivos.

---

##  Funcionalidades Principais

*   **Salas em Tempo Real:** Conexão síncrona instantânea para interação dinâmica entre o apresentador e os jogadores.
*   **Experiência Gamificada:** Sistema de pontuação dinâmico baseado em precisão e velocidade de resposta.  
*   **Painel do Host:** Controle total sobre o fluxo das perguntas, tempo limite e exibição do ranking (*leaderboard*).
*   **Criador de Quizzes:** Área administrativa intuitiva para montar perguntas, definir alternativas e gerenciar coleções.
